## What We Learned

**Technical:**
- **Exit > Entry**: You can be fast INTO a corner and still lose time. The exit determines the next straight.
- **Compression Grip**: The Uphill and Diving Turn both have compression zones that GIVE you grip. Use it.
- **Less Steering = More Speed**: Overdriving (16.9%) is costing time. Smooth inputs, better grip.

**Mental:**
- **Limit-Finding Works**: 17 incidents sounds bad. But it mapped every edge and produced a PB.
- **The Bravery Is There**: Faster than alien in scary sections. The issue isn't fear‚Äîit's follow-through.

**Strategic:**
- **Two Corners to Top 10**: T5 exit + T7 exit = 0.50s. Gap to top 10 = 0.41s. The math adds up.
